Keyboard Mapping:  

WASD – Move player
Mouse - Rotate camera
C – Walk
Left Shift – Sprint
Left Control – Crouch
Spacebar - Jump
T – Toggle Flashlight
1 – Immediate Acceleration Mode
2 – Linear Acceleration Mode
3 – Ease Acceleration Mode
B – Toggle Player Debug

Controller Mapping:

Left Stick - Move player
Right Stick - Rotate camera
LB - Walk
RB - Sprint
Right Button (Right) - Crouch
Down Button (Right) - Jump
Up Button (Right) - Toggle Flashlight
Left Button (Left) – Immediate Acceleration Mode
Up Button (Left) – Linear Acceleration Mode
Right Button (Left – Ease Acceleration Mode
Down Button (Left – Toggle Player Debug
